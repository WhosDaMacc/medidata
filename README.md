# MediData - Secure Medical Records Management System

MediData is a modern, secure, and user-friendly medical records management system that enables healthcare providers to store, share, and manage patient data efficiently.

## Features

- **Secure Patient Records**: Store and manage patient information with advanced security features
- **Medical Image Management**: Upload, view, and share medical images (MRI, CT, X-Ray, etc.)
- **Critical Data Tracking**: Monitor and manage lab results, vital signs, and diagnostic reports
- **Appointment Scheduling**: Manage patient appointments and consultations
- **Prescription Management**: Create and track prescriptions with refill management
- **Secure Sharing**: Token-based access control for secure data sharing
- **Priority Management**: Track and manage critical data with priority levels

## Tech Stack

- React with TypeScript
- Styled Components
- React Router
- Secure Token Authentication
- Modern UI/UX Design

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
